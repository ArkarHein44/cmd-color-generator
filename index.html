CMD color generator documentation

One day, I was testing the color module in the Command Prompt (CMD) on Windows. CMD supports 16 colors for both the background and the foreground. However, using the same color values for both the background and foreground does not work.

So, it's a total of 240 possible color pairs. I don't want to type color comands for 240 pairs.Then I got an idea and I want to test my classic web developnment knowledge.

To address this, I created an app to test different color pairs. The app allows you to experiment with various background and foreground color combinations.

    Disclaimer

  • I found that colors for the background and foreground need to be carefully matched, as ANSI colors used in CMD cannot be directly translated into CSS for web applications.

This app is not mobile friendly, sorry for not responsive.Thank You. !!!
